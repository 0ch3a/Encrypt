# Encrypt
```
is a open source python program that can Encrypt your source code to avoid recode and modifying.
```
# [Installation]
```
$ pkg update upgrade
$ pkg install python2
$ git clone https://github.com/0ch3a/Encrypt.git
$ cd Encrypt
```
# [Running]
```
$ python2 Encrypt.py
```
